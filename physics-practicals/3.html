Theory

A bridge rectifier is a circuit that converts alternating current (AC) to direct current (DC) using four diodes arranged in a bridge configuration. Unlike a half-wave rectifier that only utilizes half of the input waveform, a bridge rectifier utilizes both halves of the input waveform.

Working Principle

The bridge rectifier works on the principle of conducting in pairs:

  • During the positive half-cycle of AC input, diodes D1 and D3 conduct while D2 and D4 remain off
  • During the negative half-cycle, diodes D2 and D4 conduct while D1 and D3 remain off
  • This arrangement ensures that current flows through the load in the same direction for both halves of the AC cycle

Ripple Factor

The ripple factor (γ) is a measure of the effectiveness of a rectifier with filter. It is defined as:

γ = Vr / Vdc

Where:

  • Vr = RMS value of ripple voltage
  • Vdc = Average/DC value of output voltage

For a bridge rectifier without filter, the theoretical ripple factor is 0.48.

Effect of Capacitor Filter

Adding a capacitor filter to the output of a bridge rectifier:

  • Charges during the peaks of the rectified waveform
  • Discharges slowly through the load when the rectified voltage drops
  • Reduces the ripple factor significantly
  • Increases the average DC voltage level

For a bridge rectifier with capacitor filter, the ripple factor can be approximately calculated as:

γ ≈ 1 / (2√3 × f × RL × C)

Where f is the frequency of AC supply, RL is the load resistance, and C is the filter capacitance.

Experiment Procedure

Without Filter

  1. Connect the circuit as shown in the diagram (without capacitor)
  2. Apply AC input from the transformer
  3. Connect CRO across the load resistance to observe the output waveform
  4. Set the load resistance (RL) to 100Ω using the decade box
  5. Measure and record the following using CRO:
    • Volt per division
    • Time per division
    • Peak value of voltage
    • Frequency of waveform
  6. Measure the AC (RMS) and DC components of the output voltage
  7. Calculate the ripple factor as γ = Vr / Vdc
  8. Repeat the measurements for different load resistances: 100Ω, 500Ω, 1000Ω, 1500Ω, and 2000Ω

With Filter

  1. Connect a filter capacitor (4.7μF) across the load resistance
  2. Keep the load resistance fixed at 100Ω
  3. Observe the waveform on CRO and note the changes
  4. Measure the AC (RMS) and DC components of the output voltage
  5. Calculate the ripple factor
  6. Repeat the measurements for different capacitor values: 4.7μF, 10μF, 47μF, and 100μF

Key Points to Remember

1. Bridge Rectifier Basics: Uses four diodes to convert AC to DC, utilizing both halves of the AC cycle.

2. Ripple Factor: Defined as the ratio of RMS value of ripple voltage to the DC value of output voltage (γ = Vr/Vdc).

3. Without Filter: Output contains significant ripple with a theoretical ripple factor of 0.48.

4. With Filter: Adding a capacitor significantly reduces the ripple factor and smooths the output.

5. Capacitor Value: Increasing the capacitance value leads to a lower ripple factor.

6. Load Resistance: Higher load resistance with the same capacitor value gives a lower ripple factor.

7. Ripple Factor Formula: For a bridge rectifier with filter, γ ≈ 1/(2√3 × f × RL × C).
